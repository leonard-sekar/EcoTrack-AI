# EcoTrack AI

## Overview
EcoTrack AI is a Carbon Footprint Awareness Platform that helps users estimate and reduce their carbon emissions.

## Features
- Carbon Calculator
- Eco Score
- Sustainability Tips
- Responsive Design
- Input Validation

## Technologies
- HTML
- CSS
- JavaScript

## How to Run
Open index.html in a browser.

## Accessibility
- Semantic HTML
- Labels for inputs
- Keyboard accessible

## Security
- Input sanitization
- Validation
